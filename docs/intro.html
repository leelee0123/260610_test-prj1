영상 파일을 업로드하면 오디오 추출 → Whisper 자막 생성 → GPT 자막 교정 → 장면 감지 → 장면별 GPT 비전 분석 → 메타데이터 통합의 6단계 파이프라인이 자동 실행됩니다. 결과물로 원본 SRT, 교정 SRT, 장면 분석이 담긴 JSON을 제공합니다. 모든 처리 현황은 웹 페이지에서 실시간으로 확인할 수 있습니다.

OUTPUT

metadata.json — 영상 정보 + 장면별 분석
주요 기능
  • 실시간 처리 현황 표시 (2초 폴링)
  • 자막 원본/교정본 단어 단위 diff 뷰어
  • 원본·교정본 나란히 비교 (스크롤 동기화)
  • 장면 키프레임 + 분석 카드 웹 표시
  • 결과 파일 원클릭 다운로드
  • 긴 영상 자동 분할 처리 (25 MB 제한 대응)
설계 원칙
  • 타임코드 불변 — GPT에 텍스트만 전달, 타임코드는 코드가 원본에서 재조립
  • API 키 분리 — .env 보관, .env.example만 커밋
  • 단계별 폴백 — 배치 응답 불일치 시 원문 유지
  • 장면 없을 때 — 영상 전체를 1개 장면으로 처리
